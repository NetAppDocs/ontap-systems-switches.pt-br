= Instale o hardware para o switch NetApp CN1610.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Para instalar o hardware do switch NetApp CN1610, utilize as instruções em um dos seguintes guias.

* https://library.netapp.com/ecm/ecm_download_file/ECMP1117853["Guia de Instalação 1G"^].
+
Uma visão geral das características de hardware e software do switch CN1601 e do processo de instalação.

* https://library.netapp.com/ecm/ecm_download_file/ECMP1117824["Guia de Instalação 10G"^]
+
Uma visão geral dos recursos de hardware e software do switch CN1610, incluindo instruções sobre como instalar o switch e acessar a CLI.
