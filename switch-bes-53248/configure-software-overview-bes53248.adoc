= Fluxograma de instalação de software para switches BES-53248
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Para instalar e configurar o software de um switch de cluster BES-53248, siga estes passos:

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["Um"]link:configure-install-initial.html["Configure o interruptor"]
[role="quick-margin-para"]
Configure o switch de cluster BES-53248.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["Dois"]link:configure-efos-software.html["Instale o software EFOS."]
[role="quick-margin-para"]
Faça o download e instale o software Ethernet Fabric OS (EFOS) no switch de cluster BES-53248.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["Três"]link:configure-licenses.html["Instale as licenças para os switches de cluster BES-53248."]
[role="quick-margin-para"]
Opcionalmente, adicione novas portas comprando e instalando mais licenças.  O modelo básico do switch possui licença para 16 portas 10GbE ou 25GbE e duas portas 100GbE.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["Quatro"]link:configure-install-rcf.html["Instale o arquivo de configuração de referência (RCF)"]
[role="quick-margin-para"]
Instale ou atualize o RCF no switch de cluster BES-53248 e, em seguida, verifique se há uma licença adicional disponível nas portas após a aplicação do RCF.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png["Cinco"]link:configure-ssh.html["Habilite o SSH nos switches de cluster BES-53248."]
[role="quick-margin-para"]
Se você utiliza o Monitor de Saúde do Switch Ethernet (CSHM) e os recursos de coleta de logs, habilite o SSH nos switches.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-6.png["Seis"]link:reset-switch-bes-53248.html["Restaure as configurações de fábrica do switch."]
[role="quick-margin-para"]
Apague as configurações do switch do cluster BES-53248.
