= Prateleiras Cable NS224 como armazenamento conectado ao switch
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Se você possui um sistema no qual os gabinetes de unidades NS224 precisam ser cabeados como armazenamento conectado por switch (e não como armazenamento conectado diretamente), utilize as informações fornecidas aqui.

* O cabo NS224 aciona as prateleiras através dos switches de armazenamento:
+
https://library.netapp.com/ecm/ecm_download_file/ECMLP2876580["Informações para cabeamento de gavetas de unidades NS224 conectadas a switches"^]

* Instale seus switches de armazenamento:
+
https://docs.netapp.com/us-en/ontap-systems-switches/index.html["Documentação dos interruptores AFF e FAS"^]

* Confirme o hardware compatível, como switches de armazenamento e cabos, para o modelo da sua plataforma:
+
https://hwu.netapp.com/["Hardware Universe da NetApp"^]
