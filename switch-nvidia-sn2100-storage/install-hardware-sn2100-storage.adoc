= Instale o hardware para o switch NVIDIA SN2100.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
A instalação do hardware do switch NVIDIA SN2100 é um primeiro passo crucial na construção de uma infraestrutura de rede confiável e escalável para suportar fluxos de trabalho de proteção, replicação e gerenciamento de dados.

Siga estas etapas para configurar seus switches SN2100. Consulte a documentação da NVIDIA para obter instruções detalhadas.

.Passos
. Analise olink:configure-reqs-sn2100-storage.html["requisitos de configuração"] .
. Siga as instruções no link:https://docs.nvidia.com/networking/display/sn2000pub/Installation["Guia de Instalação do NVIDIA Switch"^].


.O que vem a seguir?
Após instalar o hardware, você podelink:cabling-considerations-sn2100-storage.html["revisar cabeamento e configuração"] requisitos.
